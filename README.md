# 📡 Telecom Churn — Limpeza e Preparação de Dados para Modelos de Machine Learning
📌 Sobre o Projeto
Este projeto tem como objetivo realizar a limpeza e preparação de uma base de dados real de churn de uma empresa de telecomunicações, tornando-a adequada para o treinamento de modelos de Machine Learning supervisionado.
O foco está inteiramente no processo de Data Cleaning e Feature Engineering — etapas fundamentais que precedem qualquer modelagem preditiva confiável.

🔍 O que é Churn?

Churn (ou taxa de rotatividade de clientes) é o fenômeno em que clientes deixam de fazer negócio com uma empresa — seja cancelando um serviço, encerrando uma assinatura ou migrando para um concorrente.

Em empresas de telecomunicações, o churn é um indicador crítico de negócio, pois:
* 💰 Impacta diretamente a receita — cada cliente perdido representa perda de receita recorrente
* 📉 Mede a satisfação do cliente — altas taxas de churn sinalizam problemas na experiência ou no produto
* 🔁 Custo de aquisição vs. retenção — reter um cliente é, em média, 5x mais barato do que adquirir um novo

Por isso, prever quais clientes têm maior probabilidade de sair é uma das aplicações mais valiosas de Machine Learning no setor de telecom.

🎯 Objetivo
Preparar a base de dados bruta para que ela possa ser utilizada na construção de um modelo preditivo de churn.
